# 智能垃圾分类系统简介 python714

## 项目概述

本系统是基于Python语音识别技术的智能垃圾分类平台，旨在实现便捷、高效的垃圾分类管理。系统分为前后台两部分，前后台独立运行但共用同一数据库，确保数据一致性。

## 技术栈

### 后端

- 开发语言：Python
- 框架：Django
- Python版本：3.7.7
- 数据库：MySQL 5.7+
- 数据库工具：Navicat 11+
- 开发软件：PyCharm

### 前端

- Node.js
- Vue.js
- HTML

## 功能模块

### 前台系统功能模块

1. 系统信息展示模块
    - 查看当前可回收垃圾数、各类型垃圾数及服务器信息。
2. 垃圾分类模块
    - 通过用户上传的语音进行垃圾分类。
3. 用户管理模块
    - 管理员维护和管理用户信息，包括新增和删除用户。
4. 退出功能
    - 用户通过右上角按钮注销账户。
5. 信息查看模块
    - 用户查看和修改密码。

### 后台系统功能模块

1. 垃圾分类管理模块
    - 管理员根据法律规定，对常见垃圾进行分类。
2. 用户管理模块
    - 管理员维护会员信息。
3. 系统信息管理模块
    - 修改网站首页的系统信息。

## 系统主要功能实现

### 系统首页设计

- 用户需登录后才能使用系统。
- 简洁的登录页面，包含账号、密码及登录按钮。
- 输入正确的账户密码后进入系统首页。
- 未输入账户密码时，表单验证提示。

### 系统信息展示模块

- 展示系统信息，如用户数、算法数及服务器信息。

### 基于语音识别的智能垃圾分类

- 上传或选择语音文件，点击分类按钮进行语音识别。
- 根据已有垃圾分类数据，对垃圾进行分类。
- 分类完成后，数据存储于网站。

### 用户信息维护模块

- 查看用户信息，进行新增、修改和删除操作。

- 前端：Node.js，Vue.js，HTML
